# Brick Breaker Game

Классическая игра Brick Breaker (Арканоид), разработанная с использованием JavaScript и HTML5 Canvas.

## Описание

Brick Breaker - это классическая аркадная игра, в которой игрок управляет платформой для отбивания мяча, разрушающего блоки. Цель игры - уничтожить все блоки на уровне, не дав мячу упасть вниз.

### Особенности игры

- Несколько уровней с различной сложностью
- Разнообразные типы блоков
- Система усилений (power-ups)
- Звуковые эффекты и музыка
- Система подсчёта очков

## Технологии

- JavaScript (ES6+)
- HTML5 Canvas для рендеринга
- Webpack для сборки проекта

## Установка и запуск

### Предварительные требования

- Node.js (v14.0.0 или выше)
- npm (v6.0.0 или выше)

### Установка зависимостей

```bash
npm install
```

### Запуск для разработки

```bash
npm start
```
Запустит игру в режиме разработки. Откройте браузер и перейдите по адресу http://localhost:9000.

### Сборка для продакшн

```bash
npm run build
```
Создаст оптимизированную версию игры в директории `dist`.

## Управление в игре

- **Левая/правая стрелки** или **перемещение мыши**: движение платформы
- **Пробел**: запуск мяча / пауза игры
- **ESC**: выход в меню

## Структура проекта

- `main.js` - основной файл приложения
- `entities.js` - классы игровых объектов
- `managers.js` - игровые менеджеры (звук, ресурсы и т.д.)
- `gameState.js` - управление состоянием игры
- `levels.json` - конфигурация игровых уровней
- `assets/` - игровые ресурсы (изображения, звуки)
